## 🚀 Швидкий старт

### Варіант 1: Одразу (HuggingFace)
```bash
python main.py
```
Працюватиме з HuggingFace (без ключа, але з обмеженнями)

### Варіант 2: Ollama (Рекомендується) ⭐

1. Встановіть Ollama:
   ```
   https://ollama.ai/download
   ```

2. Встановіть модель:
   ```bash
   ollama run llama3.2
   ```

3. Запустіть IDE:
   ```bash
   python main.py
   ```

### Варіант 3: LM Studio

1. Встановіть LM Studio:
   ```
   https://lmstudio.ai/
   ```

2. Завантажте будь-яку модель

3. Запустіть локальний сервер

4. Запустіть IDE:
   ```bash
   python main.py
   ```
