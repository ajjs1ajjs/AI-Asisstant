# 🚀 Local AI IDE - Повністю Локальний AI

## ✨ Особливості

- ✅ **100% Офлайн** - працює без інтернету
- ✅ **Безкоштовно** - ніяких API ключів
- ✅ **Приватно** - ваш код нікуди не відправляється
- ✅ **Розумний вибір** - показує тільки сумісні моделі
- ✅ **Все в одному** - завантаження, вибір, видалення

---

## 🚀 Встановлення

### 1. Встановіть залежності:

```bash
pip install -r requirements.txt
```

**Важливо:** `llama-cpp-python` може встановлюватися 5-10 хвилин.

Якщо виникли проблеми з встановленням:

```bash
# Спробуйте з CMake
pip install cmake
pip install llama-cpp-python --force-reinstall
```

### 2. Запустіть:

```bash
python main.py
```

---

## 🎯 Як користуватися

### Крок 1: Оберіть модель
Програма **автоматично** покаже які моделі сумісні з вашим ПК:
- 🟢 **Зелені** - сумісні, можна завантажувати
- 🔴 **Червоні** - потребують більше RAM

### Крок 2: Завантажте модель
Натисніть **⬇️ Download** біля обраної моделі
- Завантаження може тривати 5-30 хв (залежить від розміру)
- Моделі зберігаються в `~/.ai-ide/models/`

### Крок 3: Запустіть модель
Натисніть **▶️ Use This Model**
- Модель завантажиться в пам'ять
- Статус зміниться на "✅ Using: ..."

### Крок 4: Почніть спілкування
Пишіть запитання в чат - AI відповість офлайн!

## ⚙️ Налаштування продуктивності

### Якщо повільно:
1. Оберіть меншу модель (7B замість 32B)
2. Закрийте інші програми
3. Звільніть RAM

### Якщо швидко:
1. Спробуйте більшу модель
2. Збільште context size в налаштуваннях

---

## 🛠️ Вирішення проблем

### "No module named 'llama_cpp'"
```bash
pip install llama-cpp-python
```

### "Failed to load model"
- Переконайтесь що модель повністю завантажена
- Перезапустіть програму

### "Not enough memory"
- Оберіть меншу модель
- Закрийте інші програми

### Модель завантажується занадто довго
- Перевірте інтернет з'єднання
- Великі моделі (32B) можуть завантажуватися 30+ хв
