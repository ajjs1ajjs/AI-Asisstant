# 🚀 AI Coding IDE - Швидкий старт

## 1️⃣ Отримайте API ключі (БЕЗКОШТОВНО)

### Groq (найшвидший, безкоштовний тариф) ⭐ РЕКОМЕНДУЄМО:
1. Перейдіть на https://console.groq.com/keys
2. Зареєструйтесь (GitHub/Google)
3. Створіть API ключ
4. Скопіюйте ключ
5. **Безкоштовно:** 30 запитів/хвинуту

### OpenRouter (багато моделей):
1. Перейдіть на https://openrouter.ai/keys
2. Зареєструйтесь
3. Створіть API ключ
4. Поповніть баланс (мінімум $5)

### Together AI:
1. Перейдіть на https://api.together.xyz/settings/api-keys
2. Зареєструйтесь
3. Створіть API ключ
4. Отримайте безкоштовні кредити ($25)

## 2️⃣ Налаштуйте .env файл

Відредагуйте файл `.env` і вставте ваші ключі:

```env
# Достатньо одного ключа!
GROQ_API_KEY=gsk_xxxxxxxxxxxxxxxxxxxxxx
```

## 3️⃣ Запустіть IDE

```bash
python main.py
```

або двічі клікніть на `start.bat`

## 🎯 Рекомендації

Для початку використовуйте **Groq** - це:
- ✅ Безкоштовно (до 30 запитів/хв)
- ✅ Дуже швидко
- ✅ Якісні моделі (Llama 3 70B)

## ❓ Проблеми?

### Бачите "No available models":
1. Перевірте, що додали API ключ у `.env`
2. Перезапустіть програму
3. Перевірте індикатор зліва (🔌 API Status)

### Індикатор показує 🔴:
- Ключ не додано або невірний
- Перевірте `.env` файл

### Індикатор показує 🟢:
- Все працює! ✅
